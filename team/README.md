# Deep Fake Protection

## Description
This app empowers artists and privacy-conscious users to safeguard their digital identity by utilizing advanced adversarial perturbation techniques that add an invisible "cloaking" layer to images, 
fundamentally confusing AI models and preventing neural networks from accurately scraping, learning from, or recreating the image.
